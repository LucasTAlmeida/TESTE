# Sobre

A principio usando nodejs e react. Projeto de um questionário no qual você pode ver o seu número de acertos no final!


# Ferramentas Utilizadas na parte backend

-[Express]          -[React]
-[Jasonwebtoken]    
-[Mongoose]
-[BcryptJS]

# Como executar backend
```bash
$ yarn install

$ yarn dev
```
# Como executar o frontend
```bash

$ yarn install

$ yarn start
```
